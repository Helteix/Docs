Introductionâ€‹

Since the package com.ltx.tools is free, you can either install the package from my GitHub page or from the Unity Asset store.

Which should I choose?

It is recommended to stick with Asset Store's version for more stability.

Installing from the asset storeâ€‹

Before anything, make sure to get the packages here

  1. Make sure you are logged in to the same Unity account in the Unity editor that you purchased the package with.
  2. Open the Unity Package Manager (under Window/Package Manager).

  1. Look up Helteix-Tools on in the search bar and click the "install" or "download" button.

Installing from GitHubâ€‹

  1. Inside the package manager, go to the "+" button, then select the "Add from GitHub url" option

  1. Paste the https://github.com/Helteix/Tools.git url inside the text field and click on the "add" button.
